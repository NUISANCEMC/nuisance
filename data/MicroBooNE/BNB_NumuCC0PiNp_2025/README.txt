Files in this folder correspond to the "basic data release" included in the
supplemental material for Phys. Rev. D 112, 112004 (2025)
https://doi.org/10.1103/8v2y-l89l

The files bin_blocks.txt, mat_table_add_smear.txt, mat_table_cov_total.txt, and
vec_table_unfolded_signal.txt are taken directly from the published data
release.

The bin_defs.txt file provides a different representation of the binning scheme
and was taken from the original analysis inputs used with MicroBooNE's
"xsec_analyzer" tool: https://github.com/uboone/xsec_analyzer.
